# Unsichere API

## Nutzung
1. [Docker compose](https://docs.docker.com/compose/) installieren
2. `docker compose up` um die API zu starten (`--build` hinzufügen falls änderungen vorgenommen wurden)
3. die API läuft nun auf `127.0.0.1:8080`

## Abhängigkeiten
- Express
- Postgres.js
